# LifeLine Android

Production-ready Android emergency assistance application.

## Features
- SOS emergency call (112)
- Nearby hospitals map
- Offline support
- Background sync

## Tech Stack
- Kotlin + Jetpack Compose
- Hilt | Room | Retrofit | WorkManager | Google Maps

## Architecture
MVVM + Clean Architecture

## Setup
1. Add your Google Maps API key in `local.properties`
2. Replace `YOUR_API_KEY` in AndroidManifest.xml

## Demo Flow
Splash → Home → SOS / Hospital Map
